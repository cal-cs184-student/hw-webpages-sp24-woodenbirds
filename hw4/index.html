Part 5: Cloth Sim
Explain in your own words what is a shader program and how vertex and fragment shaders work together to create lighting and material effects.

A shader program is a program that render the color of each pixel on the screen. Vertex shaders are used to calculate the final position and normal vectors of each vertex. Fragment shaders are used to calculate the color of each vertex based on the output of the vertex shader.

Explain the Blinn-Phong shading model in your own words. Show a screenshot of your Blinn-Phong shader outputting only the ambient component, a screen shot only outputting the diffuse component, a screen shot only outputting the specular component, and one using the entire Blinn-Phong model.

The Blinn-Phong shading model can calculate the color of each pixel using the ambient, diffuse, and specular components.
The ambient lightings are the lightings from the environment and are the same for all pixels.
The diffuse lightings are the lightings reflected by the surface and are the same for all reflection directions.
The specular lightings are the lightings reflected by the surface and are stronger for the mirror reflection direction.
